# Latihan 1

## Masalah

Jack sangat senang dengan program penghitung total harga barang pada latihan sebelumnya. Namun sekarang Jack membutuhkan program yang dapat mengetahui produk apa yang paling banyak dibeli.

## Clue

- Method `.split`
- Method `.map`
- Method `.forEach`
- Fungsi `parseInt`
- Tanpa `while` & `for`
- Template literals (strings)
- Comparison operators

## Spesimen

**Format:**

```
<name> <sold>
```

### Test 1

**Input:**

```
gas-elpiji 20
tanggo 39
permen 2
cotton-but 60
fanta 14
```

**Output:**

```
cotton-but 60
```

### Test 2

**Input:**

```
tisu 23
teh-celup 27
garam 5
beras 89
gula 18
susu-cair 77
kardus 79
vitamin 87
```

**Output:**

```
beras 89
```
